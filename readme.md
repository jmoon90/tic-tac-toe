#[Tic Tac Toe](https://s3.amazonaws.com/tic-tac-toe/index.html)

## How to Play
+ Click on 'Who goes first?..'
+ '+' are empty spaces.
+ 'x' is computer
+ 'o' is player

## Requirements
+ It must have an unbeatable AI.  To elaborate, the computer player must
never lose and it should win when possible.
